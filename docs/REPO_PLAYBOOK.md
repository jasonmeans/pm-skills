# Repo Playbook

## Daily Use
- `scripts/new_daily_log.sh`
- Update one active skill file in `skills/`
- Log one evidence artifact

## Weekly Use
- Complete `docs/playbooks/weekly-review.md`
- Update `learnings/weekly/`
- Reprioritize next week's drills

## Monthly Use
- Fill `docs/metrics/skill-scorecard.md`
- Archive stale goals and promote effective playbooks
