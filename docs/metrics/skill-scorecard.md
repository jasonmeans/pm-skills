# Skill Scorecard

Use this monthly to measure progress by evidence, not feeling.

## Dimensions (1-5)
- Problem framing: clarity of goals, constraints, and success criteria
- System design: quality of tradeoff analysis and architecture decisions
- Implementation: correctness, readability, and maintainability
- Validation: testing depth and reliability confidence
- Delivery: planning accuracy, scope control, and follow-through

## Scoring Rule
- 1: unfamiliar, cannot execute without heavy support
- 3: can execute with moderate guidance
- 5: can lead and teach others with consistent results
